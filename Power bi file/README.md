# 💳 UPI Transactions Business Intelligence Dashboard

## 📌 Project Overview

This Power BI dashboard analyzes UPI transaction data to uncover insights into transaction trends, banking performance, customer behavior, regional adoption, and merchant categories. The dashboard supports business decision-making through interactive visualizations and KPI monitoring.

---

## 📊 Dashboard Features

- Executive KPI Dashboard
- Monthly Transaction Trend
- UPI App Performance Analysis
- Bank Performance Analysis
- State-wise Transaction Analysis
- Customer Spending Insights
- Merchant Category Analysis
- Strategic Business Recommendations
- Interactive Slicers

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📈 Key KPIs

- Total Transaction Value
- Total Transactions
- Average Transaction Value
- Success Rate
- Cashback
- Response Time

## 📌 Business Insights

- Achieved a 94.10% transaction success rate.
- BHIM recorded the highest transaction value.
- Top-performing banks processed the majority of transactions.
- Leading states contributed the highest transaction values.
- Strategic recommendations provided for improving platform performance.
